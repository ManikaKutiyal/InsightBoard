# AI Kanban Project

A full-stack AI-powered Kanban board.

## Folder Structure

frontend/            → React + Vite UI  
backend/             → Node.js + Express backend  
project-ai-kanban/   → AI logic & metadata  

## Setup Instructions

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
node server.js

### AI Module
cd project-ai-kanban
npm install
npm start
